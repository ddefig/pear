ddefig's PEAR Channel Server

Registering the channel: 
pear channel-discover ddefig.github.com/pear
Listing available packages: 
pear remote-list -c ddefig
Installing a package: 
pear install ddefig/package_name
Installing a specific version/stability: 
pear install ddefig/package_name-1.0.0
pear install ddefig/package_name-beta
Receiving updates via a feed: 
http://ddefig.github.com/pear/feed.xml
